## Introduction

The STIW3054_TheMavericks_bot is a real-time Covid-19 information tracker by searching country name.<br/> 
-It can receive country name from user and get the Covid-19 information based on the country name.<br/> 
-Later it will show the Covid-19 information to user, which are:<br/> 
*Country Name<br/> 
*Total Cases<br/> 
*Total New Cases<br/> 
*Total Death<br/> 
*Total New Death<br/> 
*Highest Cases<br/> 
*Highest New Cases<br/> 
*Highest Death Cases<br/> 
*Highest New Death Cases<br/>

## User manual
Steps:<br/> 
1.Once user open the bot link it will navigate to telegram bot .<br/>
2.User will get the information about the bot and how it will works.<br/>
3.By click the start button the bot will give instruction message to guide them with an example output.<br/>
4.The bot will ask a country name.<br/>
5.User may enter a country name to get the info about the Covid-19 cases.<br/>
6.Once the user enter the country name, the system will get the the Covid-19 information based on the country that user enter.<br/>
7.The Covid-19 information regarding to the country will be display on telegram bot interface.<br/>
8.If users want to know about other country they can enter the country name as much as they want.<br/>
